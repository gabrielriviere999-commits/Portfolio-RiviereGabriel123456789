Procédures :
Archiver les BL (bons de livraisons)
- Déverrouiller
- Cosium : cliquer sur carré bleu de favoris.
- Faire défiler tout en bas de la page.
- Cliquer sur consulter dessous achats.
- Regarder le BL qui correspond au N° et date.
- Après avoir vérifié, cliquer sur archiver.
- Faire défiler tout en haut de la page.
- CLiquer sur le rectangle archiver.
- écrire sur le papier BL : Archivé le : date d'aujourd'hui
- Mettre sur la pile BL achivés.
Ensuite ranger les montures ainsi que les étuis.

Scanner
- Déverrouiller l'ordinateur puis cliquer sur cosium.
- Regarder sur le devis le nom et le prénom de la personne.
- Regarder sa mutuelle.
- si CMU : scanner devi(s)
- si autre mutuelle (groupama, MGEN):
 - Tapper nom et prénom du client.
 - Cocher c07
 - Cliquer sur rechercher.
 - Si il y a plein de dossiers communs, cliquer sur afficher.
 - Cliquer sur la date en dessous de dossiers lunettes.
 - Regarder si il y a tous les documents
 Si il n'y a pas de prise en charge ou d'attestation CMU :
 Faire imprimer papier Améli, saisir les chiffres dessous son nom sous le NIR.
 - Cliquer sur valider.
- Prendre les papiers.
- Mettre dans l'imprimante.
- Cliquer sur munérisation.
- Appuyer sur bouton départ.
- Sur ordinateur, partir sur scan (du bureau)
- CLiquer sur le document.
- CLic droit, renommer, majuscule devis + améli ou autre + nom et prénom du client.
- Retourner sur cosium.
- Cliquer sur le petit papier.
- Cliquer sur téléchargement avancé.
- Cliquer sur téléchargement depuis disque dur.
- Type et parcourir pour sélectionner le fichier.
- Valider.
Améli = fichier client     Attestation = CMU = carte mutuelle optique     PEC = prise en charge

Réceptionner avec rvision
- Aller sur le site rvision.
- nom d'utilisateur
- Cliquer sur valider.
- Cliquer sur suivis ateliers.
- Cliquer sur réception magasin.
- Saisir le code qui est dessous le code barre du papier sur codebarre.
- Mettre les dossiers dans les péniches à facturer/appeler.
péniche : (boîte où l'on range les lunettes et autres pour les livrer au client)

Créer un rendez-vous avec minute pass
- Écrire minute dans la barre d'adresse et cliquer sur optique de bourbon.
- Cliquer sur accueil magasin.
- Cliquer sur planning.
- Cliquer sur créer un rendez-vous.
- Entrer le nom, prénom, téléphone, code postal et ville du client.
- Cliquer sur créer un nouveau client.
- Cliquer sur prestation et livraison lunettes.
- Cliquer sur collaborateur et conseiller ODB.
- Appeler le client pour connaître la date et l'heure.
Si il ne répond pas : écrire sur la fiche atelier messagerie vocale la date où on l'a appelé.
Cliquer sur fermer fermer (sans créer le rendez-vous).
Si il répond on fixe le rendez-vous et sur la fiche atelier on écrit appelé le + la date où on l'a appelé.
On clique sur la date + heure voulue et on clique sur créer le rendez-vous.
